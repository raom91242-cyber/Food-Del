# FoodDel Frontend

Responsive food delivery web application built with React, Vite, and Tailwind CSS.

## 🚀 Quick Start

### Prerequisites
- Node.js v14+
- npm or yarn

### Installation

1. **Install dependencies**
   ```bash
   npm install
   ```

2. **Start development server**
   ```bash
   npm run dev
   ```

3. **Open in browser**: http://localhost:5173

## 📱 Features

### User Features
- Browse food items by category
- Add/remove items from cart
- View cart with total
- Sign up and sign in
- Place orders
- Fully responsive design

### Admin Features
- Admin login (admin@fooddel.com / admin123)
- List/Edit/Delete items
- Add new items
- View and manage orders
- Filter orders by status

## 🎨 Responsive Design

Optimized for Desktop, Tablet, Mobile, and Small Mobile devices.

## 🔐 Routes

- `/` - Home
- `/signin` - Sign In
- `/signup` - Sign Up
- `/cart` - Cart
- `/order` - Place Order
- `/admin/login` - Admin Login
- `/admin` - Admin Dashboard

## 🚀 Build

```bash
npm run build    # Production build
npm run dev      # Development
npm run preview  # Preview build
npm run lint     # Lint code
```
